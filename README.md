### 🚀 Featured Project — Fin-Xpert

A full-stack financial management platform I built end-to-end — where my finance background meets real engineering.

- 🧾 **Invoicing** — generate, issue, and track invoices from a payment counter
- 📊 **Double-entry ledger & reports** — Balance Sheet, dashboards, institution-level aggregations
- 🔐 **Security** — MFA (TOTP) for staff accounts
- 🌍 **Internationalization** — full i18n with RTL support
- ⚡ **Performance** — short-TTL caching over heavy aggregation queries

> Built with React, Node.js, and MongoDB.
